# How to push your app to a GitHub repository

To push this project to a GitHub repository, you can use Lovable's built-in bidirectional GitHub integration. Once connected, any changes made in Lovable will automatically sync/push to GitHub, and any pushes to GitHub will automatically sync back to Lovable.

### Steps to connect and push:

1. **Open the GitHub integration menu**
   * Look at the bottom left of the chat input area in your Lovable editor.
   * Click on the **Plus (+)** icon.
   * Select **GitHub** from the menu, then click **Connect project**.

2. **Authorize Lovable on GitHub**
   * You will be prompted to authorize the Lovable GitHub App on your GitHub account.
   * Select the account or organization where you want to create the repository.

3. **Create the repository**
   * After authorization, click **Create Repository** in the Lovable interface.
   * Lovable will automatically initialize a new repository on your GitHub account and push the entire codebase of this app to it.

4. **Subsequent changes**
   * Any edits made here in Lovable are automatically committed and pushed in real-time.
   * You can also clone the repository locally, write code in your preferred IDE, and `git push` to your default branch. Those changes will automatically sync back to Lovable.
